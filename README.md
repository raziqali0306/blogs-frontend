# Blogs-Frontend

### [Visit Here](blogs-frontend.web.app)

- Note : To be able to see website, you need to allow insecure content in site settings as the backend operates over HTTP.

- Refer: Backend repo [here](https://github.com/raziqali0306/blogs-backend)

---

<br />

> ## To clone and run this repository.

<br />

1. Clone this repo using https

   ```bash
   > git clone https://github.com/raziqali0306/blogs-frontend.git
   ```

2. Go to project folder

   ```bash
   > cd blogs-frontend
   ```

3. Install the dependencies for the project using,

   ```bash
   > npm install
   ```

4. Run script command to run locally,

   ```bash
   > npm run start
   ```

---

<br />

> ## To contribute to this repo.

<br />

1. Fork this project using fork button in repo page.

1. After making changes, push your code to the forked repository

1. Open a new branch with following prefix followed by branch name

   - Note: Keep branch name relevant to the changes you made

     ```bash
     chore/<branch_name>  - changes to code
     bugfix/<branch_name> - bug fixing
     ```
